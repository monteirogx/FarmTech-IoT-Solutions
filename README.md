# 🌿 FarmTech Solutions - Irrigação Inteligente

Este projeto apresenta um sistema de monitoramento e automação para a agricultura de precisão, desenvolvido para a Fase 2 da FIAP.

## 🚀 O que o sistema faz?
O protótipo utiliza um ESP32 para tomar decisões autônomas de irrigação baseadas em:
1. **Umidade do Solo:** Aciona a bomba se a umidade for inferior a 40%.
2. **Saúde Química (pH/NPK):** Aciona a fertirrigação se o pH estiver ácido (< 6.0) e houver falta de nutrientes.
